## Overview
### Purpose
OG images are the first visual impression when Chatting content is shared on social media, Slack, Discord, or any platform that renders link previews. They must:

- **Stop the scroll** — Stand out in busy feeds
- **Communicate instantly** — Convey the content topic in <1 second
- **Reinforce brand** — Feel unmistakably Chatting
- **Drive clicks** — Create curiosity without being clickbait

### Brand Alignment
OG images extend Chatting's visual identity: warm, conversational, approachable. Not corporate. Not generic SaaS. They should feel like an invitation to a friendly conversation, not a billboard.

## Dimensions & Technical Specs
### Primary Dimensions
| Platform | Recommended Size | Aspect Ratio | Notes |
|---|---|---|---|
| Universal | 1200 × 630px | 1.91:1 | Primary format — works everywhere |
| Twitter/X | 1200 × 628px | 1.91:1 | Slight variation, 1200×630 works |
| LinkedIn | 1200 × 627px | 1.91:1 | Same as universal |
| Facebook | 1200 × 630px | 1.91:1 | Standard |
| Slack | 1200 × 630px | 1.91:1 | Shows full image |
| Discord | 1200 × 630px | 1.91:1 | Shows full image |

### Technical Requirements
| Property | Specification |
|---|---|
| File format | PNG (preferred) or JPG |
| File size | < 300KB (ideally < 150KB) |
| Color space | sRGB |
| Resolution | 72 DPI (web standard) |
| Safe zone | 48px padding from all edges |
| Text safe zone | Central 1000 × 500px area |
