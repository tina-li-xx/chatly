# Growth Strategy

## 1. Website Builders (App Marketplaces)

These marketplaces have millions of merchants actively looking for tools. Getting listed means passive, high-intent installs.

### Shopify App Store

| Step | Details |
| --- | --- |
| Build a Shopify app | Use Shopify's App Bridge plus the existing embed script. The app only needs to inject the widget code into the store theme. |
| Apply as a Partner | Create an account at [partners.shopify.com](https://partners.shopify.com/) and submit the app for review. |
| Requirements | Prepare app listing copy, screenshots, a demo store, a privacy policy, and a support contact. |
| Review timeline | Typically 2 to 4 weeks. Shopify checks functionality, UX, and policy compliance. |
| Optimize listing | Target keywords like "live chat," "customer support," and "sales chat." Get early reviews from existing Shopify customers. |

- Effort: 2 to 3 weeks of development work plus 2 to 4 weeks of review.
- Cost: Free to list. Shopify takes a 20% revenue share on apps sold through Shopify billing.

### WordPress Plugin Directory

| Step | Details |
| --- | --- |
| Package as a plugin | Wrap the embed code in a simple WordPress plugin in PHP. Add a settings page for API key and widget configuration. |
| Submit to WordPress.org | Create an account, submit the plugin via SVN, and wait for manual review. |
| Requirements | Use a GPL-compatible license, disclose any external calls, and provide a properly formatted `readme.txt`. |
| Review timeline | Usually 1 to 4 weeks because reviews are handled by volunteers. |
| Growth hack | Ask existing WordPress users to install through the directory so "Active installations" grows quickly. |

- Effort: 1 to 2 weeks of development work.
- Cost: Free.

### Webflow Marketplace

| Step | Details |
| --- | --- |
| Apply to be a partner | Apply as an App partner through [webflow.com/partners](https://webflow.com/partners). |
| Build integration | Webflow often supports a simpler model, either custom code embed or a Designer Extension API integration. |
| Create listing | Prepare professional screenshots, a clear value proposition, and an installation guide. |
| Alternative | Even without an official listing, publish a "How to add Chatting to Webflow" guide and rank for that search term. |

- Effort: 1 to 2 weeks.
- Cost: Free. Revenue share varies.

## 2. Agency/Reseller Program

Web design agencies build 10 to 50 or more sites per year. If they bundle Chatting, that creates recurring revenue at scale.

### Program Structure

| Element | Recommendation |
| --- | --- |
| Discount | 20% to 30% off retail pricing for agency partners. |
| Commission option | Alternatively, offer a 20% recurring commission on client accounts they refer. |
| Billing | Let agencies either resell directly or refer leads while Chatting bills the client. |
| White-label | On the Pro plan, agencies can remove Chatting branding. This is a strong selling point. |
| Dedicated support | Offer a partner Slack channel or a priority email path. |

### How to Find Agencies

| Method | Execution |
| --- | --- |
| Search | Google queries like "Shopify agency [city]," "WordPress development agency," and "Webflow experts." |
| Directories | Use [Clutch](https://clutch.co/), [DesignRush](https://www.designrush.com/), and [UpCity](https://upcity.com/) to find reviewed agencies. |
| LinkedIn | Search for "web design agency founder" and do personalized outreach. |
| Partner directories | Browse the public Shopify and Webflow partner directories for certified agencies. |

### Outreach Template

```text
Subject: Partner program for your clients' live chat

Hi [Name],

I noticed you build sites for [e-commerce / SaaS / local businesses]. Quick question: what do you typically recommend for live chat?

We built Chatting specifically for small teams—$29/mo vs Intercom's $500+. Takes 5 minutes to install.

We're launching a partner program: 20% recurring commission on any client you refer, or 30% discount if you want to resell directly.

Worth a 15-min call to see if it's a fit?

[Your name]
```

- Target: Reach out to 50 agencies, expect 5 to 10 responses, and aim for 2 to 3 active partners.

## 3. Review Sites (G2, Capterra, and Others)

Small business buyers search for terms like "best live chat software" and land on these sites. Reviews increase credibility and improve SEO.

### Getting Listed

| Platform | How to List |
| --- | --- |
| G2 | Create a free listing at [g2.com/products/new](https://www.g2.com/products/new) and complete the product profile. |
| Capterra | Create a free basic listing at [capterra.com/vendors](https://www.capterra.com/vendors/). Premium placement is paid. |
| Software Advice | Listing typically syncs through the same Gartner profile used for Capterra. |
| GetApp | Also part of Gartner, usually using the same shared profile. |
| TrustRadius | Create a free vendor listing at [trustradius.com/vendors](https://www.trustradius.com/vendors). TrustRadius emphasizes detailed reviews. |

### Getting Reviews

| Tactic | Details |
| --- | --- |
| Ask happy customers | After a good support moment or testimonial, ask for a quick G2 review. |
| In-app prompt | After 30 or more conversations handled, show a prompt asking active users to review Chatting. |
| Email campaign | Send a personalized request with a direct review link to the most active users. |
| Incentive | G2 and Capterra may allow gift cards for unbiased reviews, such as a $25 Amazon card, within their policy rules. |

- Target: 10 to 15 reviews to look credible. Aim for 25 or more to rank more competitively in categories.

### Optimize Your Listing

- Use real screenshots of the inbox, widget, and analytics.
- Fill out comparison and feature checklist data completely.
- Respond to positive and negative reviews promptly.
- List in the strongest relevant categories, such as Live Chat, Customer Messaging, and Help Desk if applicable.

## Priority Order

| Week | Action |
| --- | --- |
| 1 to 2 | Submit the WordPress plugin first because it is the lowest effort and serves a huge market. |
| 2 to 4 | Create G2 and Capterra listings, then begin collecting reviews. |
| 3 to 5 | Build and submit the Shopify app. |
| 4+ | Begin agency outreach at a pace of 10 emails per week. |
| Ongoing | Expand into Webflow, additional marketplaces, and review site optimization. |

## 7. Direct Outreach (for Early Traction)

| Tactic | Execution |
| --- | --- |
| Find users of competitors | Search Twitter for complaints about Intercom pricing and Zendesk complexity. DM with a genuine pitch. |
| Target specific verticals | Focus on e-commerce stores, SaaS startups, and local service businesses. Customize messaging for each. |
| Agency cold outreach | Email web design agencies: "Offer your clients live chat without the enterprise price tag." |

## 8. Churn Prevention

- Run exit surveys to understand why accounts leave.
- Send win-back emails with discounts to recently churned customers.
- Do personal outreach for high-value accounts that are at risk or have already churned.
