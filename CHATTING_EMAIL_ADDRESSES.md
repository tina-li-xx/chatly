# Chatting Email Addresses

Complete reference for all email addresses used by the Chatting app.

---

## Subdomain Structure

| Subdomain | Purpose | Volume |
|-----------|---------|--------|
| `usechatting.com` | Primary brand emails | Low |
| `mail.usechatting.com` | Visitor-facing emails | High |
| `notifications.usechatting.com` | Team notifications | Medium |

---

## `usechatting.com` — Primary Brand Emails

| Email | From Address |
|-------|--------------|
| Welcome Email | `Chatting <hello@usechatting.com>` |
| Email Verification | `Chatting <noreply@usechatting.com>` |
| Password Reset | `Chatting <noreply@usechatting.com>` |
| Team Invitation | `{{inviter_name}} via Chatting <noreply@usechatting.com>` |
| Trial Ending Reminder | `Chatting <hello@usechatting.com>` |
| Trial Expired | `Chatting <hello@usechatting.com>` |
| Product Update / Changelog | `Chatting <updates@usechatting.com>` |

---

## `mail.usechatting.com` — Visitor-Facing (High Volume)

| Email | From Address |
|-------|--------------|
| Conversation Transcript | `{{team_name}} via Chatting <noreply@mail.usechatting.com>` |
| Offline Reply Notification | `{{team_name}} via Chatting <noreply@mail.usechatting.com>` |
| Missed Chat Follow-Up | `{{team_name}} via Chatting <noreply@mail.usechatting.com>` |
| Satisfaction Survey | `{{team_name}} via Chatting <noreply@mail.usechatting.com>` |

---

## `notifications.usechatting.com` — Team Notifications

| Email | From Address |
|-------|--------------|
| New Message (Immediate) | `Chatting <noreply@notifications.usechatting.com>` |
| Daily Digest | `Chatting <digest@notifications.usechatting.com>` |

## Quick Reference

### All Addresses

| Address | Used For |
|---------|----------|

| `noreply@mail.usechatting.com` | All visitor-facing emails |
| `noreply@notifications.usechatting.com` | New messages, mentions |
| `digest@notifications.usechatting.com` | Daily digest |
| `reports@notifications.usechatting.com` | Weekly performance reports |
